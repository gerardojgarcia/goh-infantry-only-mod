This mod restricts what unit categories can be fielded in
Dynamic Conquest/Campaign by editing the conquest preset under:

  set/multiplayer/games/campaing_capture_the_flag.set

Changes:
- Only infantry and emplacement armies can be fielded. No vehicles or tanks. Airstriks can still be called in. 
- I may be able to fine tune it in the future to include light vehicles, or no artillery. or only level 1-2 artillery. 
  

Install:
- Place this mod folder in <game_root>/mods/
- Enable in: Main Menu -> Options -> Mods


Compatibility:
- Game version: 1.059.0
- Conflicts with other mods that override the same file in
  set/multiplayer/games/
